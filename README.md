# Skilltree

A project for managing and visualizing skill development paths.

## Overview

Skilltree is designed to help track and organize learning paths and skill development. This project provides a framework for creating structured learning journeys.

## Getting Started

### Prerequisites

- Modern IDE (VS Code recommended)
- Git
- GitHub Copilot (optional, but recommended for development)

### VS Code Setup

This project is optimized for development with VS Code and GitHub Copilot.

1. Install the recommended extensions when prompted, or manually install:
   - GitHub Copilot
   - GitHub Copilot Chat
   - EditorConfig

2. The workspace settings will be automatically applied when you open the project.

### Development Environment

Clone the repository:

```bash
git clone https://github.com/thinklab-architects/skilltree.git
cd skilltree
```
